login.php,register.php,logout.php tương ứng với đăng ký đăng nhập và đăng xuất
delete.php,delete_order.php tương ứng với xóa sản phẩm
detail.php là sản phẩm chi tiết
admin.php là quản lý sản phẩm,thêm hoặc xóa
cart.php là giỏ hàng
buy.php tương ứng với mua sản phâm
create_order.php thêm đơn hàng
config.php chức năng kết nối database,khởi động session,lưu thông tin và thiết lập tiếng việt
